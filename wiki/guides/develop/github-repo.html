Tools

  • manga-image-translator： 翻译图片/漫画文字的网站
  • TouhouLittleMaid： 基于Minecraft forge mod实现的东方女仆模组
  • thcrap： 基于社区实现的东方官方正作语言翻译项目
  • thtk： Touhou Toolkit
  • ReC98： 东方PC-98修复工程
  • touhou-midi-collection： 东方MIDI文件收集
  • Touhou-Web-Engine：engine for web-based bullet dodging game
  • Touhou-Public/th-music-video-generator: Touhou Project random music video generator/player, crawling image and video from websites to generate MV. (github.com)
  • wz520/thhyl: 东方STG录像信息查看器 / Touhou STG replay file(*.rpy) info viewer (github.com)
  • nyfair/foo_thbgm: ⇩Download Page⇩ A foobar2000 plugin to directly play BGMs of Touhou series and some related doujin games (github.com)
  • wz520/thkmc: Lets you use customized keymapping when playing Touhou STG by patching the games’ exe files (github.com)
  • thpatch/Universal-THCRAP-Launcher: Basic Universal Launcher for https://github.com/thpatch/thcrap
  • David-JonesDVN/Touhou-Relauncher: A remake of the Touhou Launcher by Widdiful (github.com)
  • massanoori/THRotator: THRotator allows screen rotation of Touhou Project on a vertically-oriented screen. (github.com)
  • xfgryujk/THUnpacker: A tool to unpack Touhou Project game resources (github.com)
  • MaribelHearn/maribelhearn.com: A collection of webpages for the Touhou community; largely convenient pages for the games. (github.com)
