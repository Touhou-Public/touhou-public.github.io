加拿大，不少留学的朋友都将它称作“加村”，和喧闹的国内比起来，地广人稀、绿树成荫的加拿大，确实和乡村有几分相似（另一个说法：“Canada”这个词本来就是印第安语“村落”的意思）。虽然是地广人稀，但大部分的加拿大人都生活在与美国接壤的那一片区域的城市里，所以加拿大城市的人口密度都还是尚可的，此外，亚裔人口占加拿大总人口的5%，某些城市更是高达20%以上，加之其他的一些因素，日本ACG文化在加拿大有着广泛的受众，东方Project也受益于此，越来越多的人耳濡目染的了解到了这个系列，并在进一步了解的过程中爱上了东方。

加拿大有多少东方众呢？


根据我的一个加拿大朋友Nazrinn提供的情报来看，东方众在加拿大的人数并不多，以她生活的加蒂诺来说，在那里她认识的东方众大概有5个人，并且确信有20个人知道东方Project（但不能算东方众），加蒂诺的人口有25万左右，所以如果同比例看的话，整个加拿大大概有800左右的东方众。但这个算法只能是个下限，因为加蒂诺的亚裔比例非常的低，不到0.8%（毕竟亚裔更热衷于日本ACG文化，统计资料源自维基百科），如果以亚裔占多数的地方来看，那么据我的调查及估计，加拿大的东方众上限可以达到3000~4000+。不过有个遗憾的地方，就是加拿大并没有自己的东方众聚集地，像英语圈主要活跃在美国的东方向论坛或组群里，而像法语圈的东方众，活跃的范围则更广些（下面会具体说到），所以无法通过论坛及社区的注册人数这种大致精确的方法来进行统计，我结合了这一个月来的调查结果及相关的数据，大体估算在1500~3500这个范围内。

虽然从数据上看，加拿大的东方众并不多，但是有一点需要说一下，东方开始在加拿大圈里活跃的时间并不长，许多当地的东方社团基本上都是近两年才成立起来的，但发展确实非常迅速的，我也接触到一些加拿大东方众，有相当一部分人都是12年左右才入圈的,无论是官方作品还是二次同人作品，他们非常喜欢。所以，在我看来，加拿大的东方圈是十分健康的，并且在茁壮的成长着。

他们是通过什么途径了解到东方的呢？

据我的调查，早在09年以前，就有一些人在漫展上出东方COS,不过此时他们中的大部分都是亚裔，个人认为早期绝大部分的东方爱好者都是来自亚洲的新一代移民，也许就是他们将东方从东方带入了西方，并通过各种渠道进行着传播。以我接触到的本土加拿大人来说，他们第一次接触到东方的途径基本上都是源于因特网，像YTB、Facebook，tumblr，E-hentai等等渠道。发达的社交网络使得东方Project在加拿大的ACG圈子里迅速传播开来，我观察到到，在2015年的漫展里，西方面孔的东方Coser已经占据了半边天的程度了（当然还有一些本土亚裔，但不方便区分）


东方众在加拿大的分布也和其他地方差不多，多数都生活在蒙特利尔、多伦多、渥太华、温哥华、埃德蒙顿等中型、大型城市里，不过由于加拿大国的郊区化性质，也有不少的东方众生活在城郊或者远离城市的社区、卫星城里。

这种相对集中的人口也为漫展的举办提供了合适的条件，在加拿大，每年都会举办10多起大大小小的ACG主题展，比较出名的有Otakuthon（蒙特利尔）、Anime North（多伦多）、Comiccon（渥太华）、Animethon（埃德蒙顿）、AniRevo: Winter（温哥华），


以我这个月调查的资料来看，几乎每个展子上都有东方的Cosplay,大部分展子上也能看到一些东方主题的同人摊位。几乎每个东方众都很期待与其他东方众在现实里接触的机会，所以不少人都会从相对较远的地方来参展，并结交新认识的朋友，为此，Otakuthon还专门准备了一场小型的东方交流会：“Hakurei Shrine Festival”（翻译过来就是博丽神社祭），

可以算是某种意义上的东方ONLY,人们可以在这个活动里交流东方的相关主题内容，当地同人乐队Live表演，以及进行非想天则、心绮楼等比赛等等。


由于和美国的日本ACG受众分布正好相反（美国许多漫展都开在西海岸的城市里，像洛杉矶，而东岸的漫展则相对较少，加拿大由于人口分布的原因刚好与美国相反），所以许多生活在美国东海岸的东方众也会专程参加加拿大的动漫展。


除了漫展外，加拿大的东方众在现实里很难接触到东方相关的东西，如果要买东方物的话，基本上都是通过ebay等电子商务网站来海淘（里面有个有趣的现象就是你能在ebay上看到很多天朝的东方同人物，我猜测这些基本上都是海外倒爷弄的，虽然我对这种现象也进行过一定深入的调查，不过我想把这个留在美国篇里再个各位细细的道一道），加拿大人告诉我华人街上的一些杂志店会卖一些盗版的东方印刷品（也是天朝货，比方说东方主题的P站画集等等），虽然知道是盗版，但苦于购买东方物的渠道太少（ebay上其实也不多），所以一些人会购买这类东西来解解馋。虽然很不幸，不过情况也在慢慢好转中，已经有组织正在启动面向欧美的东方同人物购物网站：HakureiCon Shop（http://hakureiconshop.nazrinndev.com），

也许不久的将来他们就能很方便的买到海外东方物了吧。

他们有没有什么东方主题论坛呢？

由于领国的过于发达，许多事变的不那么重要了，比方说拥有一个自己国家的东方主题论坛。因为没有自己专门的聚集地，大部分加拿大东方众都通过社交网站作为获取情报或与他人交流的主要场所，除了社交网络外，不少人也混迹于英语向东方论坛里，像Maidens of the Kaleidoscope (https://www.shrinemaiden.org)、gensokyo(http://www.gensokyo.org/)等等，在魁北克等以法语为主的地区里，东方众则有着更广的社交环境，除了英文论坛外，他们也常去法语向东方论坛，像Touhou-Online(http://touhou-online.net/)等。


（许多加拿大人都混迹于国外的东方论坛，像Touhou-Online和Maidens of the Kaleidoscope）


对加拿大留学圈东方众状况的调查也是我的一个重点方向。我个人倒是很想了解下加拿大华人圈或者留学圈里东方的状况，但经过数番努力，依旧没有得到这方面的合适情报，以我目前了解到的情况来看，许多留学的东方众并没有融入当地的圈子里，原因很多，比方说信息渠道不足，比方说不好意思勾搭等等，但这只是部分，据我在FB及微博上观察到一些在加留学的人来说（估计他们大概都不怎么混天朝的东方圈），他们倒是都有很好的融于当地的东方圈里，比方说Facebook上“東方 Touhou Project Fans [Alberta] ”组，虽然是埃德蒙顿本地东方交流组，但这个组的管理员就是一位天朝的留学生，也比方说我在FB观察的加拿大东方圈里的各个的人脉关系，就有不少华裔（从名字的姓氏能看出来）都有标注来自天朝的XX省（虽然不清楚是移民还是留学）。


很遗憾，虽然有一定的了解，到目前为止我所搜罗的这方面情报远远达不到我的预期，所以我希望有在加拿大留学的朋友看到这块后能帮助我补完这块的内容，感谢。


COSPLAY作品是观察一个ACG作品很好的渠道，我也通过手头上的各种渠道和方法获得了相当丰富的内容，加拿大的COS圈发展的很不错，每届漫展上都有很多出东方COS的东方众，也许是西方特有的娱乐至死精神吧，也有不少人出的“我这个角色有点特殊”，不拘形式，高兴就好。

加拿大人非常热爱艺术。作为当代艺术创作之国，加拿大有着许多大大小小的社团及艺术团体，这其中就有不少的东方众，而东方二次创作，则给予了他们充裕的空间来享受创作带来的乐趣。

加拿大有一些东方同人社团，并且进行着同人作品的创作。除了我之前给大家介绍的Ziyang Yip Films

2016年Ziyang Yip Films的年初新作）

（详情请参考：http://tieba.baidu.com/p/4298801047）

还有其他的一些像以现场Live演奏为主的Yokai Project（http://www.yokai-project.com/，注：2015年已解散）


，其中社员Yannick在社团解散后又创立了一个东方主题的同人音乐社团Lunatyc Records（http://lunatycrecords.bandcamp.com/），


另一个音乐社团Kawaii Bass，（http://kawaiibass.com/）则每年都会在多伦多的酒吧里举行数场DJ舞会


除以上，加拿大还有一些个人社团，比方说Sedrife

（http://sedrife.bandcamp.com/），就是一个很厉害的同人创作家，因为是位韩国移民，除了参加加拿大本地漫展外，他也时常参加韩国的同人展，并贩售自己的同人作品（我会在合适的时机里在欧美情报站系列里给大家详细的介绍下他）

还有像同人绘画家ofSkySociety，也制作过东方同人作品，除了东方外，她也广泛涉猎于ACG题材作品。


（ofSkySociety的东方作品，http://ofskysociety.deviantart.com/art/Touhou-Reimu-Hakurei-520885768）

此外，加拿大还有很多其他的同人物创作者，通过厂家提供的模板等方式，自己设计相关的同人物，并在漫展上进行贩售，此外，一些绘画爱好者也会将自己的绘画拿到漫展上出售。


（感谢atrican对我情报的纠正）

还有一个我需要单独说一下，Touhou Project X GS System（https://www.kickstarter.com/projects/18592227/touhou-project-ft-gs-system），一个由加拿大组织GSSystem所设计的东方桌游，并在kickstarter上通过众筹的手段进先行发售，这个众筹得到了不少人的支持，很快就达到了他们制定的众筹金额目标。他们的目标是开发一个泛用的卡牌系统，用东方题材打头阵，但基于对东方同人性质的顾虑，在未能申请到ZUN的许可之后，他们取消了Touhou Project X GS System的众筹，高质量的东方同人作品就这样流产了，也是一种遗憾。

（接近完成品的Touhou Project X GS System，现已被终止）


以上就是目前我所调查到的全部内容，不过World,Touhou系列并不是以简单的介绍就完事为目标而创立的，我希望本系列能带给大家一个广阔的眼界来洞察这个广阔的世界，所以本帖到目前为止只能算是个开始，如果有合适的消息，我会将这个贴一直更下去，我也欢迎各位能在这个系列里分享自己在外国的东方趣闻趣事，


One Touhou,One Dream!


个人评价及补充：

1、Hakurei Shrine Festival活动止步于2019年，但官方并未出终止公告，所以未来的情况有待观察；

2、虽然各国东方圈都出现了各种程度上的缩水，但加拿大的情况似乎还好，缩减比例不是很大，加拿大漫展上依然有一些东方cos，但近期的照片主要以亚裔Coser为主，可能是我看这方面内容比较匆忙导致的；

3、文章里提到的HakureiCon Shop已经试运营了，可以谷歌到；

4、我印象里这篇文章之后加拿大还有出现过几个东方社团和相关作品，但当时没有进行资料留存，一时半会找不到；

5、各种原因，当地网络上的椛梗玩的似乎比之前更6了。
